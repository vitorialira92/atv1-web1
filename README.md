## Sobre o Projeto

Este projeto foi desenvolvido para a disciplina **Desenvolvimento de Software para a Web 1** da Universidade Federal de São Carlos, ministrada pelo **Prof. Delano M. Beder**.

O objetivo da atividade foi utilizar as tecnologias fundamentais da web com o apoio do framework Bootstrap.

Para este trabalho, desenvolvi um **site de portfólio pessoal**, personalizando tanto o conteúdo quanto o estilo visual para refletir minha identidade.

---

## Template Utilizado

O template utilizado foi o Grayscale, template gratuito do site Start Bootstrap. Acesse o template aqui: https://startbootstrap.com/theme/grayscale
